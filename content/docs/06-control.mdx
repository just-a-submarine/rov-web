## 一、Xbox 手把（有線 USB HID）

### 連接方式

有線連接延遲低、穩定，**不佔用無線電資源**，避免與 ESP-NOW + Wi-Fi 競爭 2.4GHz。BLE 備案但不建議（BLE + ESP-NOW + Wi-Fi 三者同時跑，無線電資源競爭嚴重）。

- 函式庫：Library Manager 搜尋 `ESP32_USB_Host_HID`
- 硬體：Xbox 手把 USB 線接至地面站 ESP32-S3 的 USB Type-A OTG 轉接頭

### 按鍵對應

| 按鍵 | 功能 | 備註 |
|------|------|------|
| 左搖桿 | 水平推進（前後 + 轉向） | 自動模式下鎖定 |
| 右搖桿 Y 軸 | 深度控制 | 永遠有效（包含自動模式） |
| Y 鍵 | LED 開關（最高優先） | toggle |
| RB 鍵 | 串流模式切換（0 ↔ 1） | 邊緣觸發 |
| LB 鍵 | 手動拍照（720p 存 SD） | 邊緣觸發，模式 0 才有效 |
| Start 鍵 | 緊急停車 | 三馬達歸零 + EN 斷開 |

### 地面站搖桿讀取與指令打包

```cpp
#include <ESP32USBHost.h>

ESP32USBHost usbHost;

void setupXbox() {
    usbHost.begin();
}

void readXbox() {
    usbHost.task();

    int lx = usbHost.getAxisX(0);  // -32768 ～ +32767
    int ly = usbHost.getAxisY(0);
    int ry = usbHost.getAxisY(1);

    bool btnY     = usbHost.getButton(BTN_Y);
    bool btnRB    = usbHost.getButton(BTN_RB);
    bool btnLB    = usbHost.getButton(BTN_LB);
    bool btnStart = usbHost.getButton(BTN_START);

    // RB 邊緣觸發：模式 0 / 1 切換
    static bool rbLast = false;
    if (btnRB && !rbLast) streamMode = (streamMode == 0) ? 1 : 0;
    rbLast = btnRB;

    // LB 邊緣觸發：模式 0 才送出拍照指令
    static bool lbLast = false;
    bool takePhoto = false;
    if (btnLB && !lbLast && streamMode == 0) takePhoto = true;
    lbLast = btnLB;

    // 死區處理
    if (abs(lx) < 2000) lx = 0;
    if (abs(ly) < 2000) ly = 0;
    if (abs(ry) < 2000) ry = 0;

    // 映射至馬達值（-1023 ～ +1023）
    int fwd   = map(ly, -32767, 32767, 1023, -1023);
    int turn  = map(lx, -32767, 32767, -1023, 1023);
    int depth = map(ry, -32767, 32767, 1023, -1023);

    // 差速混控（詳見「一、差速轉向演算法」章節）
    MotorPair mp = computeDifferential(lx, ly);
    int leftVal  = mp.left;
    int rightVal = mp.right;

    // 打包 ControlPacket
    ControlPacket pkt;
    pkt.leftMotor     = leftVal;
    pkt.rightMotor    = rightVal;
    pkt.vertMotor     = depth;
    pkt.ledOn         = ledState;
    pkt.emergencyStop = btnStart;
    pkt.streamMode    = streamMode;
    pkt.takePhoto     = takePhoto;
    pkt.msgType       = 0;
    esp_now_send(rov_sta_mac, (uint8_t*)&pkt, sizeof(pkt));
}
```

## 一之一、差速轉向演算法（Differential Steering）

### 概念說明

本 ROV 採**差速轉向**（Tank Drive / Differential Steering）：左右馬達速度差形成轉彎，速度和形成前進推力，**不設實體舵面**。

| 搖桿輸入 | 左馬達 | 右馬達 | 結果 |
|----------|--------|--------|------|
| 純前進（↑） | +V | +V | 直線前進 |
| 純右轉（→） | +V | −V | 原地右旋 |
| 斜右前（↗） | +V | 0 | 右弧前進 |
| 斜左前（↖） | 0 | +V | 左弧前進 |

### 設計目標

1. **搖桿推的大小決定速度**：搖桿推越遠 → 速度越快；輕推 = 慢速微調
2. **搖桿推的角度決定轉向程度**：越偏側邊 → 轉彎越急；越偏前後 → 越接近直線

### 演算法：極座標歸一化差速混合

標準 `left = fwd + turn` 做法有一個問題：斜向推滿搖桿時輸出超過 1.0 而被 constrain 截斷，導致「斜角速度比直線速度慢」。改良版使用**歸一化**：讓輸出峰值永遠等於搖桿實際大小（magnitude），覆蓋全方向角度。

```
數學推導：

  x = lx 正規化（右 = +1）
  y = −ly 正規化（前進 = +1，搖桿 Y 軸天生反向）

  r = clamp( sqrt(x²+y²), 0, 1 )    ── 搖桿大小（速度命令）
  
  rawLeft  = y + x
  rawRight = y − x
  peak     = max(|rawLeft|, |rawRight|)

  若 peak > ε：
    leftPow  = rawLeft  × (r / peak)
    rightPow = rawRight × (r / peak)
  否則：
    leftPow = rightPow = 0

  → 輸出之最大絕對值 ≡ r（速度由搖桿大小決定）
  → 輸出之比例 ≡ tan(θ)（轉彎由搖桿角度決定）
```

### 完整實作程式碼

```cpp
#include <cmath>

#define DEADZONE_RATIO 0.06f   // 死區（~2000/32767）

struct MotorPair { int left; int right; };

/**
 * 極座標歸一化差速轉向
 * 輸入：lx, ly ∈ [-32767, +32767]（Xbox 搖桿原始值，ly 向前為負）
 * 輸出：left, right ∈ [-1023, +1023]（BTS7960 PWM 值）
 */
MotorPair computeDifferential(int lx, int ly) {
    // 1. 正規化至 [-1, 1]；Y 軸反向（搖桿前推 ly 為負 → 前進為正）
    float x =  (float)lx / 32767.0f;
    float y = -(float)ly / 32767.0f;

    // 2. 死區
    if (fabsf(x) < DEADZONE_RATIO) x = 0.0f;
    if (fabsf(y) < DEADZONE_RATIO) y = 0.0f;

    // 3. 計算搖桿大小（速度命令），最大值限制為 1.0
    float r = fminf(hypotf(x, y), 1.0f);

    // 4. 差速混合
    float rawLeft  = y + x;
    float rawRight = y - x;

    // 5. 歸一化：讓輸出峰值等於搖桿大小 r（保持速度比例正確）
    float peak = fmaxf(fabsf(rawLeft), fabsf(rawRight));
    float leftPow, rightPow;
    if (peak > 1e-6f) {
        leftPow  = rawLeft  * (r / peak);
        rightPow = rawRight * (r / peak);
    } else {
        leftPow = rightPow = 0.0f;
    }

    // 6. 轉換至 PWM 整數範圍
    return {
        constrain((int)(leftPow  * 1023.0f), -1023, 1023),
        constrain((int)(rightPow * 1023.0f), -1023, 1023)
    };
}
```

### 行為驗證表（搖桿推滿）

| 搖桿方向 | x | y | rawLeft | rawRight | peak | left | right | 說明 |
|----------|---|---|---------|----------|------|------|-------|------|
| 純前（↑） | 0 | 1 | 1 | 1 | 1 | +1023 | +1023 | 全速前進 |
| 純後（↓） | 0 | −1 | −1 | −1 | 1 | −1023 | −1023 | 全速後退 |
| 純右（→） | 1 | 0 | 1 | −1 | 1 | +1023 | −1023 | 原地右旋 |
| 斜右前 45°（↗）| 0.71 | 0.71 | 1.41 | 0 | 1.41 | +1023 | 0 | 右弧前進 |
| 輕推前（50%）| 0 | 0.5 | 0.5 | 0.5 | 0.5 | +511 | +511 | 半速前進 |
| 輕推斜右（50%）| 0.35 | 0.35 | 0.71 | 0 | 0.71 | +511 | 0 | 半速右弧 |

> **關鍵特性**：
> - 斜角推滿（0.71, 0.71）輸出 (+1023, 0)，而非被截斷後的 (+1023, 0)，速度等同直線推滿 → 全方向 360° 都能發揮完整速度
> - 輕推任意方向，輸出成比例縮小 → 操作手感線性、精確

## 二、控制流向

### 手動操控

```
Xbox 左搖桿 → 差速混控 → ControlPacket → ESP-NOW → ROV
    ↓
BTS7960 RPWM/LPWM → 左右馬達

Xbox 右搖桿 Y 軸 → ControlPacket → ROV → 垂直馬達（永遠有效）
```

### 自動導航

```
手機 Leaflet 地圖點擊航點 → HTTP POST → 地面站
    ↓ ESP-NOW 分塊傳送
ROV 接收航點 → Potential Field + Geofence 逐點導航
    水平馬達：自動控制（左搖桿鎖定）
    垂直馬達：右搖桿永遠有效
```

## 三、馬達 PWM 控制

```cpp
#define PWM_FREQ  20000   // 20kHz，超出可聞範圍
#define PWM_RES   10      // 10-bit，0～1023

void setupMotorPWM() {
    ledcAttach(14, PWM_FREQ, PWM_RES);  // 左馬達 RPWM
    ledcAttach(21, PWM_FREQ, PWM_RES);  // 左馬達 LPWM
    ledcAttach(47, PWM_FREQ, PWM_RES);  // 右馬達 RPWM
    ledcAttach(43, PWM_FREQ, PWM_RES);  // 右馬達 LPWM
    ledcAttach(44, PWM_FREQ, PWM_RES);  // 垂直馬達 RPWM
    ledcAttach(46, PWM_FREQ, PWM_RES);  // 垂直馬達 LPWM
}

// speed: -1023（全速反轉）到 +1023（全速正轉）
void setMotor(uint8_t rpwmPin, uint8_t lpwmPin, int speed) {
    speed = constrain(speed, -1023, 1023);
    if (speed >= 0) {
        ledcWrite(rpwmPin, speed);
        ledcWrite(lpwmPin, 0);
    } else {
        ledcWrite(rpwmPin, 0);
        ledcWrite(lpwmPin, -speed);
    }
}
```

## 四、MCP23017 初始化與安全啟動

```cpp
#include <Wire.h>
#include <Adafruit_MCP23X17.h>

Adafruit_MCP23X17 mcp;

void setupMCP() {
    Wire.begin(1, 2);        // SDA=GPIO1, SCL=GPIO2
    mcp.begin_I2C(0x20);     // A0/A1/A2 全接 GND

    // GPA0～GPA6 全設輸出，預設 LOW
    // EN 全關（馬達不動）、繼電器高電位觸發所以 LOW = 不動作（LED 不亮）
    for (int i = 0; i <= 6; i++) {
        mcp.pinMode(i, OUTPUT);
        mcp.digitalWrite(i, LOW);
    }
}

void enableMotors() {
    // 所有初始化完成後才呼叫，拉高 EN 啟用馬達電路
    for (int i = 0; i <= 5; i++) mcp.digitalWrite(i, HIGH);
}

void emergencyStop() {
    // PWM 全歸零
    ledcWrite(14, 0); ledcWrite(21, 0);
    ledcWrite(47, 0); ledcWrite(43, 0);
    ledcWrite(44, 0); ledcWrite(46, 0);
    // EN 全斷
    for (int i = 0; i <= 5; i++) mcp.digitalWrite(i, LOW);
}
```

## 五、自動導航演算法

### Potential Field + Geofence

- **吸引力**：目標航點對 ROV 產生吸引力，力的方向指向航點
- **排斥力**：Geofence 邊界對 ROV 產生排斥力，防止偏出安全區域
- **合力**：兩者向量相加，轉換為左右馬達差速值

### GPS 精度限制

- Neo-M8N CEP 精度約 **2.5 公尺**
- **最小航點間距：≥ 5 公尺**（否則 GPS 誤差會導致航點判定混亂）
- **航點到達半徑：4 公尺**（距航點 < 4m 視為到達，切換下一個航點）
- 手機 Leaflet UI 設置航點時應提示用戶保持足夠間距

### 航向來源：QMC5883P 羅盤

#### 硬鐵／軟鐵校正

有刷直流馬達的永磁體與運轉電流會產生恆定與變動磁場，若不校正，羅盤讀數可能偏差數十度。

**校正流程**：

1. **硬鐵校正（靜態偏移）**：馬達靜止，將 ROV 水平旋轉 360°，記錄 X/Y 軸最大值與最小值，計算偏移量 `offsetX = (maxX + minX) / 2`，`offsetY = (maxY + minY) / 2`
2. **軟鐵校正（軸向比例）**：計算 X/Y 軸半徑比 `scaleX / scaleY`，修正橢圓為圓形
3. **馬達干擾補償**：分別在馬達靜止與運轉時取樣，建立 PWM 值對應的磁場偏移查表，導航時動態補償

```cpp
float getCorrectedHeading() {
    sensors_event_t event;
    compass.getEvent(&event);

    // 硬鐵校正
    float x = event.magnetic.x - offsetX;
    float y = event.magnetic.y - offsetY;

    // 軟鐵校正
    x *= scaleX;
    y *= scaleY;

    // 馬達干擾補償（簡化版：根據當前 PWM 查表減去偏移）
    x -= motorCompX[currentPWMLevel];
    y -= motorCompY[currentPWMLevel];

    float heading = atan2(y, x) * 180.0 / PI;
    if (heading < 0) heading += 360.0;
    return heading;
}
```

## 六、PID 深度控制

使用 QuickPID 函式庫，控制垂直馬達維持目標深度。

```cpp
#include <QuickPID.h>

float depthInput, depthOutput, depthSetpoint;
QuickPID depthPID(&depthInput, &depthOutput, &depthSetpoint);

void setupDepthPID() {
    depthPID.SetTunings(2.0, 0.5, 0.1);        // Kp, Ki, Kd（需實測調整）
    depthPID.SetOutputLimits(-1023, 1023);       // 對應馬達 PWM 範圍
    depthPID.SetSampleTimeUs(100000);            // 100ms = 10Hz
    depthPID.SetMode(depthPID.Control::automatic);
}

void updateDepthControl() {
    depthInput = ms5837.depth();                 // 當前深度（公尺）
    depthPID.Compute();
    setMotor(44, 46, (int)depthOutput);          // 垂直馬達
}
```

右搖桿永遠有效：右搖桿輸入值直接覆蓋 PID 輸出，放開搖桿後恢復 PID 控制。
