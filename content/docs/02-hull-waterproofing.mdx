## 艇體結構

### 三段式設計

| 段落 | 材質                      | 說明                           |
| ---- | ------------------------- | ------------------------------ |
| 前端 | PP 端蓋（現成品）         | 純封閉端蓋，無任何開孔或安裝物 |
| 中段 | PETG 列印（Bambu Lab A1） | 艇體主體，所有內容物均安裝於此 |
| 後端 | PP 端蓋（現成品）         | 純封閉端蓋，無任何開孔或安裝物 |

### 結構細節

- **PS 半球觀察窗**：嵌入 PETG 中段頂部開孔，OV5640 置於半球後方取像
- **尾翼**：直接裝於 PETG 中段
- **重心**：電池置艙底，搭配配重塊微調，維持微正浮力
- **馬達電線**：水平馬達線由機翼延伸臂根部穿線孔（D=6mm，沿 X 軸徑向穿入艙內）進入艙室；垂直馬達線由馬達座底部引線孔（D=6mm）進入艙室；各穿線孔以環氧樹脂灌封防水
- **垂直馬達**：置於船底圓柱形馬達座（OD=38mm，高 33mm），馬達線從底部引線孔（D=6mm）進入艙室

### 外殼開孔（共 2 個）

| 孔號   | 用途                                                                       | 位置          | 密封方式                   |
| ------ | -------------------------------------------------------------------------- | ------------- | -------------------------- |
| 頂部孔 | PS 半球嵌入（OV5640 觀察窗）                                               | PETG 中段頂部 | 接縫 C（環氧樹脂黏合收邊） |
| 孔 1   | MS5837 訊號線 ＋ GPS 天線線（主動式）＋ ESP32 天線（IPEX→SMA，3m SMA 線） | PETG 中段側面 | 熱熔膠堵底 ＋ 環氧樹脂灌封 |

## 防水方案

### 接縫與密封點總覽

| 編號       | 位置                                        | 接合材質          |
| ---------- | ------------------------------------------- | ----------------- |
| 接縫 A     | 前端 PP 蓋 ↔ PETG 中段                     | PP ↔ PETG        |
| 接縫 B     | 後端 PP 蓋 ↔ PETG 中段                     | PP ↔ PETG        |
| 接縫 C     | PS 半球 ↔ PETG 中段頂部                    | PS ↔ PETG        |
| 穿孔 1     | MS5837 訊號線 ＋ GPS 天線線 ＋ ESP32 SMA 線 | PETG 孔壁         |
| 馬達穿線孔 | 馬達動力線（每顆馬達 2 條，共 6 條）        | 機翼臂根部 D=6mm × 2 ＋ 垂直馬達座引線孔 D=6mm × 1 |

### 接縫 A / B 密封工序（PP ↔ PETG）

1. **表面處理**：接縫兩側以砂紙打磨，去除油脂；PP 面塗佈 PP 專用底料，靜置活化
2. **專用膠黏合**：塗佈雙劑型 PP 專用膠（雙液混合），貼合夾緊，完全固化
3. **自融膠帶（主防水層）**：拉伸 1.5～2 倍纏繞，重疊率 50%，纏繞兩層
4. **PVC 絕緣膠帶（防磨損）**：自融膠帶外側纏一層，純防磨損
5. **環氧樹脂塗佈**：外露 PETG 外皮整體薄塗多次，填補列印層間孔隙

### 接縫 C 密封工序（PS ↔ PETG）

1. 打磨兩側（PS 半球法蘭面與 PETG 孔緣）
2. 塗佈低黏度雙液環氧樹脂，貼合固定，完全固化
3. 外圈補一層環氧樹脂收邊

### 穿孔 1 灌封工序

1. 三條線纜穿過孔洞，調整至最終位置
2. 艙內側打入熱熔膠堵底
3. 艙外側灌入低黏度雙液環氧樹脂，分次少量，填滿間隙
4. 靜置完全固化（建議 24h 慢固化型）
5. 外緣補一層環氧樹脂收邊

### 馬達穿線孔灌封工序

馬達動力線由機翼延伸臂根部穿線孔（D=6mm）進入艙內（左右水平馬達各 1 個），垂直馬達線由馬達座底部引線孔（D=6mm）進入艙內，共 3 個穿線孔。

1. 馬達電線穿入引線孔，調整至最終位置
2. 孔外側灌入低黏度環氧樹脂，分次少量填滿間隙
3. 孔內側艙室面同樣補封
4. 靜置完全固化（建議 24h 慢固化型）

### MS5837 艙外密封工序

感測孔內有凝膠保護膜，**感測孔須完全暢通，不得塗佈任何材料**。

- PCB 背面與線材焊點整體灌低黏度環氧樹脂，僅保留感測孔面暴露
- PCB 背面與艇體金屬件之間墊熱縮管，防電偶腐蝕
- 感測孔朝向：朝下或側面，避免正對水流

### 防水驗證流程

| 步驟     | 方法                                                             |
| -------- | ---------------------------------------------------------------- |
| 氣壓測試 | 充入 0.3 bar，肥皂水檢查所有接縫與穿線孔，靜置 15 分鐘無洩壓 |
| 淺水浸泡 | 水桶靜置 30 分鐘，開蓋確認內部無水氣                             |
| 深度測試 | 實際潛水至最大工作深度，回收後開蓋確認                           |

## 浮力估算

### 元件重量估算

| 項目                                          | 估算重量                |
| --------------------------------------------- | ----------------------- |
| 動力電池（3S4P，12 顆 18650 + 保護板 + 外殼） | ~600 g                  |
| 馬達 × 3（含螺旋槳）                         | ~450 g                  |
| BTS7960 × 3                                  | ~90 g                   |
| ESP32-S3-CAM + 感測器 + MCP23017 + 繼電器     | ~120 g                  |
| PETG 殼體（中段，含馬達座、機翼臂、尾翼）     | ~120 g                 |
| PP 端蓋 × 2                                  | ~80 g                   |
| PS 半球                                       | ~20 g                   |
| 系統電池 + 升壓模組                           | ~60 g                   |
| 配線、密封材料、雜項                          | ~100 g                  |
| **小計（未含配重）**                    | **~1600–1650 g** |

### 浮力需求

- 總重約 1600–1650 g（含配重前）；艇體排水 ≈ 1590 cm³（≈ 1590 g 浮力）
- 淨浮力 **≈ −0 g（接近中性浮力）**；需加裝少量外部浮力補償以達到微正浮力
- 設計目標：微正浮力（淨浮力 +50～200 g），緊急停車時可自動上浮
- 用配重塊（鉛塊或螺帽）微調至目標浮力
- **PETG 中段尺寸：直徑 9 cm × 長 25 cm（250 mm），壁厚 1.5 mm**（PP 端蓋 OD ≈ 90mm，壁厚 0.85mm）
  - 外壁排水體積：π × 4.5² × 25 ≈ **1590 cm³**
  - 接近中性浮力，**需加裝少量外部閉孔泡棉浮力補償**（≈ 100 cm³ 以上）
  - 25 cm（250 mm）長度在 Bambu Lab A1 列印尺寸（256 mm）限制內（僅餘 6mm），可**一體成型**不需分段
