## 一、外殼尺寸規格

### 整體尺寸

| 項目 | 數值 | 說明 |
|------|------|------|
| 外徑（OD） | **90 mm** | 對齊 PP 端蓋外徑 |
| 壁厚 | **1.5 mm** | 約 3 層壁圈（0.4mm 噴嘴）|
| 內徑（ID） | **87 mm** | OD − 2 × 壁厚 |
| 總長 | **250 mm** | 一體成型，Bambu A1 256mm 限制內 |
| PP 端蓋插入槽深（每端） | **10 mm** | 前後各一，PP 蓋插入後外表面齊平 |
| 主艙有效長度 | **230 mm** | 250 − 10 × 2 |

### PP 端蓋來源

PP 端蓋取自市售圓柱形 PP 保鮮盒底部切開取蓋（錐形容器，越靠近開口外徑越大）：

| 位置 | OD | 壁厚 |
|------|-----|------|
| 窄端（底部） | ≈ 86.7 mm | 0.85 mm |
| **寬端（本設計採用）** | **≈ 90 mm** | 0.85 mm |

建議精確量測切割高度處的實際 OD，依實測值微調插槽尺寸。

### 座標系

- **Z 軸**：艇體長軸（Z=0 後端 / Z=250 前端）
- **Y 軸**：垂直方向（Y+ 為艇頂）
- **X 軸**：水平橫向（X+ 為右舷）
- **原點**：艇體後端面中心

```
後端(Z=0)  ←────────── 250 mm ──────────→  前端(Z=250)
    ┌──────────────────────────────────────────┐
    │←10mm→      主艙有效艙室      ←10mm→ │
    │PP 插槽  （有效長 230 mm）  PP 插槽   │
    └──────────────────────────────────────────┘
                  OD = 90 mm，壁厚 1.5 mm
```

## 三、Bambu Studio 切片參數

### Quality（品質）

| 參數 | 建議值 | 說明 |
|------|--------|------|
| Layer height | **0.16 mm** | 層高越薄，層間融合越好，防水性提升 |
| Initial layer height | 0.20 mm | 第一層稍厚有助附著 |
| Outer wall 線寬 | **0.45 mm** | 略加寬填滿層間微縫 |
| Inner wall 線寬 | **0.46 mm** | 同理 |
| Smart scarf seam | ✅ 開啟 | 避免 Z seam 留縫，對防水關鍵 |

### Strength（強度）

| 參數 | 建議值 | 說明 |
|------|--------|------|
| **Wall loops** | **4** | 最關鍵：1.5mm ÷ 0.45mm ≈ 3.3 條，設 4 讓壁圈緊密壓實 |
| **Detect thin wall** | **✅ 開啟** | 必開！壁厚 1.5mm 屬薄壁，不開會出現未填充縫隙 |
| Top shell layers | **7** | 頂面密實（配合 0.16mm 層高） |
| Bottom shell layers | **5** | 底面加厚 |
| **Sparse infill density** | **100%** | 防水件必須 100% 填充 |
| Sparse infill pattern | Concentric | 100% 填充時對圓柱體密封性最佳 |
| Infill/Wall overlap | **25%** | 消除壁-填充接縫微縫 |

### Speed（速度）

| 參數 | 建議值 | 說明 |
|------|--------|------|
| **Outer wall** | **80 mm/s** | 慢速讓塑料充分融合 |
| **Inner wall** | **120 mm/s** | 同理降速 |
| Small perimeters | **≈ 32 mm/s** (40%) | 細節處密合 |
| Top surface | **100 mm/s** | 確保封頂密實 |
| Sparse infill | **150 mm/s** | 100% 填充不宜太快 |
| Initial layer | **30 mm/s** | 確保牢牢附著 |

| Type | Tree (auto) |

### Others（其他）

| 參數 | 建議值 |

| 參數 | 建議值 | 說明 |
|------|--------|------|

## 四、防水效果優先順序

1. **Wall loops 2→4 + Detect thin wall 開啟** — 消除 1.5mm 壁內空隙（最關鍵）
2. **Infill 15%→100%** — 消除內部空腔
3. **噴嘴溫度 240–245°C + 風扇降至 30–50%** — 提升層間黏合力
4. **Layer height 0.2→0.16mm** — 增加層間密封面積
5. **外壁速度 200→80 mm/s** — 讓塑料充分融合
6. **Infill/Wall overlap 15%→25%** — 消除壁-填充接縫

## 五、預估規格對比

| 項目 | 原設定 | 防水設定 |
|------|--------|----------|
| 預估列印時間 | 約 3～4 小時 | 約 8～10 小時 |
| 外壁強度 | 低（2 圈） | 高（4 圈，近實心壁） |
| 防水性能 | 差（壁間有縫） | 佳（全融合） |

## 六、印完後驗證

1. **目視檢查**：外壁應無可見縫隙或透光點
2. **浸水測試**：密封兩端後完全浸入水中，觀察是否冒泡
3. **加壓測試**（選用）：從穿線孔注水加壓至 0.1 bar，保壓 5 分鐘無滲漏
