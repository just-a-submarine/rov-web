## 任務定義

- **通訊距離**：約 20–200 公尺（操作員沿岸步行跟隨）
- **深度**：20 cm～1 公尺（由浮標拖纜長度決定）

## 材料清單

### 主控與電子

| 項目       | 規格                                                       |
| ---------- | ---------------------------------------------------------- |
| ROV 主控   | ESP32-S3-CAM（Freenove N16R8，16MB Flash + 8MB OPI PSRAM） |
| 地面站主控 | ESP32-S3-CAM（Freenove N16R8）                             |
| GPS        | Neo-M8N（模組體艙內，主動式天線 LNA 28dB 至浮標）          |
| 電子羅盤   | QMC5883P（I²C 0x2C）                                      |
| 水壓感測器 | MS5837-02BA（I²C 0x76）                                   |
| 電流計     | INA260（I²C 0x40）                                        |
| IO 擴展板  | MCP23017（I²C 0x20）                                      |
| 馬達驅動板 | BTS7960 × 3                                               |
| MicroSD 卡 | 32GB                                                       |
| 繼電器模組 | 單路 12V，設為**高電位觸發**                         |
| 升壓模組   | 5V/2A × 1（ROV 系統電路用；地面站無升壓模組）             |

### 艇體結構

| 項目         | 規格                              |
| ------------ | --------------------------------- |
| 主體（中段） | PETG 列印（Bambu Lab A1）         |
| 前蓋 / 後蓋  | PP 端蓋（現成品）× 2             |
| 鏡頭視窗     | PS 透明半球                       |

| 配重塊       | 適量（微調至微正浮力 +50～200 g） |

### 密封材料

| 項目               | 用途                                               |
| ------------------ | -------------------------------------------------- |
| PP 專用底料        | 接縫 A/B PP 面活化                                 |
| 雙劑型 PP 專用膠   | 接縫 A/B 第一道黏合                                |
| 自融膠帶           | 接縫 A/B 主防水層                                  |
| PVC 絕緣膠帶       | 接縫 A/B 防磨損保護                                |
| 低黏度雙液環氧樹脂 | 接縫 C、穿孔灌封、馬達穿線孔灌封、MS5837 灌封、PETG 外皮 |
| 熱熔膠             | 穿孔灌封輔助堵底                                   |
| 熱縮管             | MS5837 電氣隔離                                    |

| 項目             | 規格                                         |
| ---------------- | -------------------------------------------- |

| GPS 天線         | 主動式（LNA 28dB），3m 延長線                |
| 浮標外殼         | PET 防水盒 + 泡棉                            |
